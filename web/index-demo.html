🏗️ ExpoBeton RDC - Chatbot Widget

✅ Widget actif!

Le chatbot est maintenant disponible en bas à droite de votre écran.

Cliquez sur le bouton flottant 💬 pour démarrer une conversation!

📋 Installation sur votre site

Pour intégrer ce chatbot sur n'importe quelle page de votre site, ajoutez simplement cette ligne avant la balise </body>:

<!-- ExpoBeton RDC Chatbot Widget -->
<script src="https://expobetonrdc.com/chat/chat-widget-standalone.js"></script>

🎯 Fonctionnalités

  • ✅ Bouton flottant discret
  • ✅ Formulaire de contact initial
  • ✅ Réponses intelligentes sur ExpoBeton RDC
  • ✅ Tracking des conversations
  • ✅ Emails automatiques à bot@expobetonrdc.com
  • ✅ Auto-fermeture après 10 min d'inactivité
  • ✅ Compatible mobile et desktop
  • ✅ Ne nécessite aucune configuration côté serveur

🔧 Configuration

Le widget est configuré pour pointer vers votre backend PythonAnywhere.

Important: Modifiez l'URL dans chat-widget-standalone.js ligne 16:

const RASA_SERVER_URL = 'https://VOTRE_USERNAME.pythonanywhere.com';

📦 Fichiers à uploader sur votre serveur

  1. chat-widget-standalone.js - Le script du widget
  2. index-demo.html - Cette page de démo (optionnel)

Uploadez ces fichiers dans un dossier /chat/ sur votre hébergement partagé.

🌐 Intégration WordPress

Si votre site est sous WordPress:

  1. Allez dans Apparence → Personnaliser → CSS additionnel
  2. Ou ajoutez dans Apparence → Éditeur de thème → footer.php avant </body>
<?php if (!is_admin()) : ?>
<script src=""></script>
<?php endif; ?>
